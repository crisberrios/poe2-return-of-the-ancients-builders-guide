> Master list of every patch change that materially affects an existing build. Per-ascendancy files reference this so we don't re-evaluate the same change 23 times.
>
> **Excluded by user instruction:** the Runes of Aldur league mechanic, new runes/runeforging mechanics, brand-new Lineage Supports that exist only for the new league, the Atlas tree "Origins of Divinity" overhaul. Mentioned in passing only when they directly affect an existing skill/item.
>
> Direction: **+** buff · **−** nerf · **~** neutral / clarity · **★** new content. Magnitude: **L/M/S** = large/medium/small.

## How to read this matrix in per-ascendancy files

Each file picks the subset of these rows that apply to its ascendancy's typical skills/items/passive tree picks, then rates direction/magnitude in that ascendancy's column. We never re-evaluate "what's S31 doing" — only "does S31 apply to this ascendancy."
